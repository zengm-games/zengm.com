God Mode enables some features that are a little too close to cheating for me to want enabled by default. Specifically:

  • The ability to create new players (this feature has been around for a while, but now it is only available in God Mode)
  • The ability to edit players (available only since earlier today)
  • The ability to force trades to be accepted (brand new)

To enable God Mode within a league, go to Tools > God Mode. The only caveat is that if you enable God Mode in a league, you'll no longer record any achievements for that league, for the obvious reason that God Mode makes it really easy to cheat. Even if you turn off God Mode, the achievements ban will persist.

If you notice any bugs in God Mode or you have ideas for other things to include in God Mode, please post here, on Reddit, on Facebook, on Twitter, or email me.
